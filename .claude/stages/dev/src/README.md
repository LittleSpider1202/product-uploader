# 源代码 (src/)

## 存放内容
- 项目源代码
- 配置文件
- 测试代码
- Prompt模板

## 目录结构建议

```
src/
├── main/              # 主程序代码
├── config/            # 配置文件
├── prompts/           # Prompt模板（AI项目）
├── tests/             # 测试代码
└── scripts/           # 脚本工具
```

## 代码规范
- 遵循项目技术方案中定义的规范
- 保持代码简洁，避免过度设计
- 编写必要的注释和文档字符串

## 测试要求
- 核心功能需有单元测试
- AI相关功能需有效果评测
