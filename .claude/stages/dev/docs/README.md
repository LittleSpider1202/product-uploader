# 研发文档 (docs/)

## 存放内容
- 技术方案文档
- 架构设计图
- API文档
- 部署文档

## 文件命名规范
- `tech-design-v{版本号}.md` - 技术方案
- `architecture-{模块名}.md` - 架构设计
- `api-{模块名}.md` - API文档
- `deploy.md` - 部署文档
